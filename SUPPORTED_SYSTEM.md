# Supported environments

Notepad++ Web runs in modern browsers and does not depend on a specific desktop OS for end users.

## Browser support target

- Latest stable Chrome
- Latest stable Edge
- Latest stable Firefox
- Latest stable Safari

## Notes

- Functionality depends on standard browser APIs (DOM, local storage, clipboard, downloads).
- Mobile browser support is best-effort and may vary by OS/browser combination.
- For development/build support, see [BUILD.md](BUILD.md).
