# Notepad++ Web

Notepad++ Web is a browser-based text editor inspired by the Notepad++ experience.

> This project is an independent web adaptation and is **not** affiliated with, endorsed by, or connected to the official Notepad++ project or its author.

## Version

Current MVP target version: **1.3.0**

## Quick start

```bash
cd web
npm install
npm run dev
```

Then open `http://localhost:3000`.
